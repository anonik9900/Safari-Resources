Apple File System.

Documents, photos, emails, apps: these are all important data for you, and it is up to your Mac's file system
to organize them into files and folders that you can open with one click The current file system was designed
together with the first Mac and has always behaved very well, but today the flash technology in our
computers opens new horizons of innovation. It's time to set a new foundation for the future:

with macOS High Sierra, on every Mac with flash storage comes Apple File System,
which thanks to an advanced architecture ensures a new level of responsiveness and security.
